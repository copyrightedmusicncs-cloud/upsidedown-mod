# Upside Down Mod - Placeholder Textures

These are placeholder textures that need to be replaced with proper 16x16 pixel art.

## Required Textures

| `corrupted_grass_top.png` | Blue-purple grass with organic veins (top view) |
| `corrupted_grass_side.png` | Blue-purple grass on top, dark dirt below |
| `corrupted_dirt.png` | Dark purplish-brown soil |
| `corrupted_stone.png` | Dark grey stone with red veins |
| `dead_grass.png` | Brown wilted grass cross texture |
| `dead_flower.png` | Grey/brown withered flower |
| `dead_leaves.png` | Brown/grey decayed leaves (transparent) |

## Color Palette Suggestions

### Corrupted Grass
- Base: `#3a2d5c` (deep purple-blue)
- Highlights: `#4a3d7c` (lighter purple)
- Veins: `#2a1d4c` (dark purple)

### Corrupted Stone
- Base: `#3a3a3a` (dark grey)
- Veins: `#8a2a2a` (dark red)

### Dead Vegetation
- Base: `#5a4a3a` (brown)
- Dead: `#4a4040` (grey-brown)

## Tools for Creating Textures
- [Piskel](https://www.piskelapp.com/) - Free online pixel art editor
- [Aseprite](https://www.aseprite.org/) - Professional pixel art tool
- [GIMP](https://www.gimp.org/) - Free image editor

## Texture Template
Each texture should be exactly 16x16 pixels, PNG format with transparency where needed.
